# 토큰/컨텍스트 관리 규칙

## 코드 크기 제한
- **파일 하나**: 최대 200줄, 초과 시 모듈 분리
- **함수 하나**: 최대 50줄
- 큰 작업은 단계별로 나눠서 진행

## 컨텍스트 사용량 구간별 행동 기준

| 구간 | 행동 |
|------|------|
| 0~50% | 자유롭게 작업 |
| 50~70% | 큰 파일 첨부 자제 |
| 70~85% | `/compact` 실행 권장 |
| 85~95% | `/compact` 즉시 실행 |
| 95% 이상 | `/clear` 후 새 대화 시작 |

## 보고 규칙
- 매 **10번 응답**마다 `/status` 결과 보고
- 이미 작성된 코드 **재출력 금지** (파일명 + 함수명만 언급)

## 출력 효율
- 코드 변경 시 **diff 형식** 우선
- 불필요한 장문 설명 생략
- 실제 코드와 구조 중심으로 답변
