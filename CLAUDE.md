# 맛집 블로그 자동화 시스템 - Claude Code 설정

## 언어 규칙
- 모든 답변은 **한국어**로 작성
- 코드 주석도 **한국어**로 작성

## 프로젝트 개요
- **프로젝트명**: 맛집 블로그 자동화 시스템
- **기술스택**: Python, Streamlit, OpenAI ChatGPT API, 네이버 키워드도구 API, 네이버 데이터랩 검색어트렌드 API
- **목표**: 음식점 정보 입력 → 키워드 분석 → SEO 최적화 블로그 글 자동 생성

## 작업 시작 전 필수
- `docs/` 폴더 전체 읽기 (CONTEXT.md, SKILLS.md, TOKEN.md, API.md)

## API 키 관리
- API 키는 반드시 `.env`에서만 읽기
- 코드에 API 키 **하드코딩 절대 금지**

## LLM 정책
- 텍스트 생성은 **OpenAI ChatGPT API만** 사용
- Anthropic Claude, Gemini 계열 **사용 금지**

## 출력 규칙
- 이미 작성된 코드를 다시 출력하지 말 것 → **파일명 + 함수명**만 언급
- 변경사항은 **diff 형식** 우선
- 토큰 절약을 위해 불필요한 설명 생략, 코드 위주로 답변

## 컨텍스트 관리
- 컨텍스트 70% 초과 시 `/compact` 실행
- 매 10번 응답마다 `/status` 결과 보고

## 코드 품질
- 파일 하나 최대 200줄, 초과 시 모듈 분리
- 함수 하나 최대 50줄
- 한 파일에 모든 로직 몰아넣지 말 것
